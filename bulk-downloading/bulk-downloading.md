# Bulk Downloading

## Available APIs

| API                 | Explanation                                   |
| ------------------- | --------------------------------------------- |
| `startForeground()` | Start a download in the foreground            |
| `startBackground()` | Start a download in the background            |
| `check()`           | Check the number of tiles in a certain region |
| `cancel()`          | Cancel any ongoing downloads                  |
